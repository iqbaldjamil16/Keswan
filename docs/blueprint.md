# **App Name**: PuskeswanPro

## Core Features:

- Service Entry: Input form for date, officer name, owner name/address, iSIKHNAS case ID, livestock type/count, clinical symptoms, diagnosis, handling, treatment type, and treatment details.
- Data Storage: Store inputted healthcare service data to a database.
- Reporting: Generate reports based on stored healthcare service data.
- Data Search: Allows searching all the stored data

## Style Guidelines:

- Primary color: Light green (#A7D1AB), reminiscent of veterinary environments.
- Background color: Very light green (#F0F4F1). A desaturated, bright tint of the primary for a calming background.
- Accent color: Blue (#9AC6C5) for interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font that balances modernity with warmth.
